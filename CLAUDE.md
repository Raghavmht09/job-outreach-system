## Project: Job Outreach System

### Overview
A job outreach application to automate searching, contact discovery, message customization, and tracking.

### Tech Stack
- **Frontend**: Next.js 14+ (App Router), Tailwind CSS
- **Backend**: Express (Node.js) with TypeScript
- **Database/Auth**: Supabase
- **Service Layer**: RESTful APIs

### Commands
- **Frontend**:
  - `npm run dev`: Start development server
  - `npm run build`: Build for production
- **Backend**:
  - `npm run dev`: Start development server (nodemon)
  - `npm run build`: Compile TypeScript

### Directory Structure
- `frontend/`: UI code (Next.js)
- `backend/`: API server (Express)
- `services/`: Core service logic
- `shared/`: Shared types and utilities
- `docs/`: Documentation

### Coding Standards
- Use TypeScript for all new code.
- Functional components with Hooks for React.
- Service layer for business logic.
- RESTful API design patterns.
